# Theme Toggle


<h3>LIVE DEMO:- https://md-raquib-hussain90.github.io/Theme-Preference/ </h3>
A simple and responsive Dark/Light Theme Toggle built with HTML, CSS, and JavaScript.

## Features

- 🌙 Dark Mode Support
- ☀️ Light Mode Support
- 💾 Theme Persistence using LocalStorage
- 🖥️ Automatic System Theme Detection
- 🔄 Real-Time Theme Updates when System Theme Changes
- ⚡ Lightweight and Easy to Integrate

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)

## How It Works

1. Detects the user's system theme preference.
2. Applies the appropriate theme on page load.
3. Saves the selected theme in LocalStorage.
4. Restores the saved theme on future visits.
5. Updates the theme instantly using the toggle button.
